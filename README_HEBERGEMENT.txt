Installation sur serveur
========================

1) Ouvrir le dossier public_html ou www de votre hebergement.
2) Mettre directement le CONTENU de cette archive dans public_html / www.
   Ne mettez pas le dossier BiotechAgro_site_vitrine comme sous-dossier.
3) Le fichier index.html doit rester a la racine du serveur.
4) Le site s'ouvrira automatiquement avec : www.BiotechAgro.com
5) Le fichier .htaccess redirige /index.html vers / pour ne pas afficher index.html dans l'adresse.
6) Le formulaire contact envoie vers : biotechagro.digital@gmail.com si PHP/mail est active sur le serveur.

Important : le nom de domaine www.BiotechAgro.com doit etre configure chez votre hebergeur/DNS.
Le fichier du site ne peut pas creer le domaine a lui seul.
